⭐ 1. Order Request Lifecycle

⭐ 2. UI Screens for Order Requests

2.1 Order Request List

Shows all requests.

• Search → <input type="text">
• Filters → <select>
• Status → <select>
• Table → <table>


---

2.2 Create New Order Request

User chooses:

• Chemical OR Instrument


HTML Elements

Label	Element	Notes	
Request Type	<select>	Chemical / Instrument	
Chemical Name	<select>	Only if chemical	
Instrument Name	<select>	Only if instrument	
Quantity	<input type="number">	Required	
UOM	<select>	mg / g / mL / L / unit	
Required By Date	<input type="date">	Optional	
Reason for Request	<textarea>	Required	
Supplier	<select>	Optional	
Attachments	<input type="file">	Optional	
Submit Button	<button>		


---

2.3 Order Request Details Screen

Shows:

• Request info
• Status
• Approval history
• Delivery tracking


HTML Elements

UOM	<span>

2.4 Approve Order Request

Approver sees:

2.5 Order Placement Screen

Label	Element
